# 例题01: 轨道最高点临界条件

## 题目内容
如图所示，半径为 R 的圆周轨道固定于竖直面内，轨道内侧光滑，一质量为 m 的小球在轨道内侧做圆周运动，小球可视为质点，重力加速度为 g，经过轨道最高点时刚好不脱离轨道。

**问题**: 小球在最高点时，下列说法正确的是（ ）

A. 小球的线速度大小等于 √(gR)
B. 轨道对小球的压力等于 mg
C. 小球的线速度大小等于 0
D. 小球所需的向心力大于重力

## 物理模型
- **模型类型**: 绳模型（无支撑轨道）
- **运动类型**: 竖直平面内的圆周运动
- **临界状态**: 刚好不脱离轨道

## 受力分析

### 最高点受力
1. **重力**: 竖直向下，大小为 mg
2. **轨道压力**: 垂直于轨道表面，指向圆心（可能为零或正值）

### 向心力方程
在最高点，向心力由重力和轨道压力的合力提供：
\[ mg + N = \frac{mv^2}{R} \]

## 临界条件分析

### "刚好不脱离轨道"的含义
当小球刚好不脱离轨道时，轨道对小球的支持力为零：
\[ N = 0 \]

### 临界速度计算
代入向心力方程：
\[ mg = \frac{mv_{min}^2}{R} \]
\[ v_{min} = \sqrt{gR} \]

## 选项分析

### A. 小球的线速度大小等于 √(gR)
- **正确** ✅
- 这是刚好不脱离轨道的临界速度

### B. 轨道对小球的压力等于 mg
- **错误** ❌
- 临界状态下 N = 0，不等于 mg

### C. 小球的线速度大小等于 0
- **错误** ❌
- 如果 v = 0，小球会掉落，无法完成圆周运动

### D. 小球所需的向心力大于重力
- **错误** ❌
- 临界状态下，向心力等于重力：F_向 = mg

## 解题要点

### 1. 理解"刚好不脱离"
- 轨道压力 N = 0
- 向心力完全由重力提供
- 速度达到最小值 v_min = √(gR)

### 2. 速度范围
- v > √(gR): 正常圆周运动，N > 0
- v = √(gR): 临界状态，N = 0
- v < √(gR): 无法到达最高点，会脱离轨道

### 3. 能量关系
从最低点到最高点，机械能守恒：
\[ \frac{1}{2}mv_{low}^2 = \frac{1}{2}mv_{high}^2 + 2mgR \]
代入 v_high = √(gR):
\[ v_{low} = \sqrt{5gR} \]

## 常见错误

### 错误1: 混淆绳模型和杆模型
- **绳模型**: 最高点必须有 v ≥ √(gR)
- **杆模型**: 最高点可以 v = 0

### 错误2: 认为压力等于重力
- 在最高点，压力可能小于、等于或大于重力
- 具体取决于速度大小

### 错误3: 忽略向心力的方向
- 向心力始终指向圆心
- 在最高点，重力和压力都指向圆心

## 扩展思考

### 1. 如果速度大于临界速度
当 v > √(gR) 时：
\[ N = m\left(\frac{v^2}{R} - g\right) > 0 \]
轨道提供额外的向心力。

### 2. 如果速度小于临界速度
小球无法到达最高点，会在到达最高点前脱离轨道，做斜抛运动。

### 3. 实际应用
- 过山车设计
- "水流星"杂技
- 摩托车"飞车走壁"

## 练习建议

1. **计算不同位置的速度**
   - 已知最高点速度，求最低点速度
   - 已知最低点速度，判断能否完成圆周运动

2. **分析压力变化**
   - 绘制压力随位置变化的图像
   - 分析最大压力和最小压力出现的位置

3. **能量守恒应用**
   - 结合机械能守恒解决问题
   - 考虑摩擦损耗的情况

## 总结
- **正确答案**: A
- **关键公式**: v_min = √(gR)
- **物理意义**: 重力单独提供向心力的临界状态
- **解题关键**: 理解"刚好不脱离"意味着 N = 0
