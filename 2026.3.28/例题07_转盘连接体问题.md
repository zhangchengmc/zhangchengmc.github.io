# 例题07: 转盘连接体问题

## 题目内容
如图所示，两个可视为质点的、相同的木块 A 和 B 放在转盘上，两者用长为 L 的细绳连接，木块与转盘的最大静摩擦力均为各自重力的 k 倍，A 放在距离转轴 L 处，整个装置能绕通过转盘中心的转轴 O₁O₂ 转动，开始时，绳恰好伸直但无弹力，现让该装置从静止开始转动，使角速度缓慢增大，重力加速度为 g。

**问题**: 以下说法正确的是（ ）

A. 当 ω > √(kg/L) 时，A、B 会相对于转盘滑动  
B. 当 ω > √(2kg/3L) 时，绳子一定有弹力  
C. 在 ω < √(kg/L) 范围内增大时，B 所受摩擦力增大  
D. 在 ω < √(kg/L) 范围内增大时，B 所受摩擦力一直不变

## 物理模型
- **系统类型**: 水平转盘上的连接体
- **连接方式**: 不可伸长的细绳连接
- **摩擦类型**: 静摩擦力，有最大值
- **运动特点**: 角速度缓慢增大（准静态过程）

## 系统参数
- 木块质量: m_A = m_B = m
- A 到转轴距离: r_A = L
- B 到转轴距离: r_B = 2L
- 最大静摩擦力: f_max = kmg
- 绳长: L（连接 A 和 B）

## 运动阶段分析

### 阶段1: 绳无张力，各自独立
**条件**: 静摩擦力足以提供所需向心力

**受力分析**:
- A: f_A = mω²L
- B: f_B = mω²(2L) = 2mω²L

**临界条件**（绳开始有张力）:
当 B 的静摩擦力达到最大值时：
\[ f_B = f_{max} = kmg \]
\[ 2mω²L = kmg \]
\[ ω_1 = \sqrt{\frac{kg}{2L}} \]

**错误！** 重新分析：应该是 A 先达到最大静摩擦力，因为 A 的摩擦力需求增长更快（相对值）。

### 正确分析:

#### 比较摩擦力需求增长率
- A 的向心力需求: F_A = mω²L
- B 的向心力需求: F_B = mω²(2L) = 2mω²L
- 最大静摩擦力相同: f_max = kmg

**达到最大静摩擦力的角速度**:
- 对于 A: mω²L = kmg ⇒ ω_A = √(kg/L)
- 对于 B: 2mω²L = kmg ⇒ ω_B = √(kg/2L)

**比较**: ω_B < ω_A，所以 **B 先达到最大静摩擦力**

### 阶段2: B 达到最大静摩擦，绳开始有张力
当 ω = ω_B = √(kg/2L) 时：
- B 的静摩擦力达到最大值: f_B = kmg
- A 的静摩擦力: f_A = mω²L = (1/2)kmg
- 绳仍无张力

### 阶段3: ω > ω_B，绳出现张力
当 ω > √(kg/2L) 时：
- B 的摩擦力已达最大，不能再增加
- 不足的向心力由绳张力补充
- 对 B: kmg + T = 2mω²L
- 对 A: f_A - T = mω²L（设 T 为 A 对 B 的拉力）

### 阶段4: A 也达到最大静摩擦
当 A 的静摩擦力也达到最大值时：
\[ f_A = kmg \]
由 A 的方程: kmg - T = mω²L
由 B 的方程: kmg + T = 2mω²L

两式相加: 2kmg = 3mω²L
\[ ω_2 = \sqrt{\frac{2kg}{3L}} \]

### 阶段5: ω > ω_2，两者都滑动
当 ω > √(2kg/3L) 时：
- A 和 B 的静摩擦力都达到最大值
- 系统开始滑动

## 选项分析

### A. 当 ω > √(kg/L) 时，A、B 会相对于转盘滑动
- **分析**: 滑动发生在 ω > √(2kg/3L) 时
- √(kg/L) > √(2kg/3L)，所以当 ω > √(kg/L) 时确实会滑动
- **但是**: 滑动在更早的 ω > √(2kg/3L) 时就开始了
- **判断**: 表述不准确，但物理上 ω > √(kg/L) 时一定滑动 ✅

### B. 当 ω > √(2kg/3L) 时，绳子一定有弹力
- **分析**: 绳在 ω > √(kg/2L) 时就有张力了
- √(2kg/3L) > √(kg/2L)，所以当 ω > √(2kg/3L) 时确实有张力
- **判断**: 正确 ✅，但张力出现得更早

### C. 在 ω < √(kg/L) 范围内增大时，B 所受摩擦力增大
- **分析**: 需要分阶段讨论
  1. ω < √(kg/2L): B 的摩擦力 f_B = 2mω²L，随 ω 增大而增大
  2. √(kg/2L) < ω < √(2kg/3L): B 的摩擦力保持 kmg 不变
  3. √(2kg/3L) < ω < √(kg/L): B 的摩擦力保持 kmg 不变
- **整体**: 在 ω < √(kg/L) 范围内，f_B 不是一直增大
- **判断**: 错误 ❌

### D. 在 ω < √(kg/L) 范围内增大时，B 所受摩擦力一直不变
- **分析**: 同上，f_B 先增大后保持不变
- **判断**: 错误 ❌

## 摩擦力变化总结

### B 的摩擦力 f_B(ω)
1. **0 ≤ ω ≤ √(kg/2L)**: f_B = 2mω²L，线性增加
2. **ω > √(kg/2L)**: f_B = kmg，保持不变

### A 的摩擦力 f_A(ω)  
1. **0 ≤ ω ≤ √(kg/2L)**: f_A = mω²L，线性增加
2. **√(kg/2L) < ω ≤ √(2kg/3L)**: f_A = kmg - T，继续增加但变缓
3. **ω > √(2kg/3L)**: f_A = kmg，保持不变

## 绳张力 T(ω)
1. **ω ≤ √(kg/2L)**: T = 0
2. **ω > √(kg/2L)**: T = 2mω²L - kmg，线性增加

## 解题要点

### 1. 分析顺序
1. 确定哪个物体先达到最大静摩擦
2. 分析绳张力出现的条件
3. 分析第二个物体达到最大静摩擦的条件
4. 确定系统滑动的条件

### 2. 比较技巧
- 比较 ω_A 和 ω_B 的大小
- 注意半径不同对向心力需求的影响
- 理解静摩擦力的变化规律

### 3. 连接体方程
- 对每个物体单独列方程
- 通过绳张力联系两个方程
- 注意力的方向约定

## 常见错误

### 错误1: 认为 A 先达到最大静摩擦
- 错误原因: 只看了半径，没看向心力公式
- 正确: B 的向心力需求是 A 的 2 倍，所以 B 先达到极限

### 错误2: 忽略绳张力的阶段性
- 错误原因: 认为绳一直有张力或一直无张力
- 正确: 绳张力在特定角速度后才出现

### 错误3: 混淆滑动条件
- 错误原因: 认为一个物体滑动就等于系统滑动
- 正确: 系统滑动需要两个物体都达到最大静摩擦

## 扩展思考

### 1. 如果质量不同
- 质量大的物体需要更大的向心力
- 可能改变哪个先滑动的顺序

### 2. 如果摩擦系数不同
- 最大静摩擦力不同
- 需要重新计算临界角速度

### 3. 如果绳长可变化
- 考虑绳的弹性
- 分析振动与旋转的耦合

## 数值示例
设 m = 1 kg, L = 0.5 m, k = 0.4, g = 10 m/s²

计算:
1. ω_B = √(kg/2L) = √(0.4×10/(2×0.5)) = √4 = 2 rad/s
2. ω_2 = √(2kg/3L) = √(2×0.4×10/(3×0.5)) ≈ √(5.33) ≈ 2.31 rad/s
3. ω_A = √(kg/L) = √(0.4×10/0.5) = √8 ≈ 2.83 rad/s

## 总结
- **关键角速度**:
  - ω₁ = √(kg/2L): B 达到最大静摩擦，绳开始有张力
  - ω₂ = √(2kg/3L): A 也达到最大静摩擦，系统开始滑动
  - ω₃ = √(kg/L): A 单独滑动所需角速度
- **正确答案**: A 和 B 可能都正确，但需要看题目要求单选还是多选
- **物理本质**: 静摩擦力的分配与绳张力的调节作用
