# 圆周运动公式总结

## 一、基本运动学公式

### 1.1 线量与角量关系
| 物理量 | 公式 | 说明 |
|--------|------|------|
| 线速度 v | v = ωr | r: 半径，ω: 角速度 |
| 角速度 ω | ω = 2π/T = 2πf | T: 周期，f: 频率 |
| 向心加速度 a | a = ω²r = v²/r | 方向指向圆心 |
| 周期 T | T = 2πr/v = 2π/ω | 完成一周的时间 |
| 频率 f | f = 1/T = ω/2π | 单位时间转数 |

### 1.2 各量之间的关系
\[
v = \omega r, \quad \omega = \frac{2\pi}{T}, \quad a = \frac{v^2}{r} = \omega^2 r
\]

## 二、动力学公式

### 2.1 向心力公式
| 情境 | 公式 | 说明 |
|------|------|------|
| 通用公式 | F = ma = mω²r = mv²/r | 向心力 |
| 水平圆周 | F = f（静摩擦力） | 靠静摩擦力提供 |
| 竖直圆周（最高点）| F = mg ± N | "+": 压力，"-": 拉力 |
| 圆锥摆 | F = mg tanθ | θ: 绳与竖直方向夹角 |

### 2.2 不同模型的向心力

#### 2.2.1 绳模型（无支撑）
\[
F = 
\begin{cases}
mg + T & \text{(最高点，T向下)} \\
T - mg & \text{(最低点，T向上)} \\
\sqrt{(mg)^2 + (mv^2/r)^2} & \text{(任意点)}
\end{cases}
\]

#### 2.2.2 杆模型（有支撑）
\[
F = 
\begin{cases}
mg ± N & \text{(最高点，N可正可负)} \\
N - mg & \text{(最低点，N > mg)}
\end{cases}
\]

#### 2.2.3 内外轨道模型
\[
F = 
\begin{cases}
mg - N & \text{(外轨道最高点)} \\
mg + N & \text{(内轨道最高点)}
\end{cases}
\]

## 三、临界条件公式

### 3.1 水平转盘上的物体
| 条件 | 公式 | 说明 |
|------|------|------|
| 最大静摩擦力 | f_max = μN = μmg | μ: 摩擦系数 |
| 临界角速度 | ω_c = √(μg/r) | 开始滑动的角速度 |
| 临界线速度 | v_c = √(μgr) | 开始滑动的线速度 |

### 3.2 竖直平面内的圆周运动

#### 3.2.1 绳模型（最高点）
| 条件 | 公式 | 说明 |
|------|------|------|
| 最小速度 | v_min = √(gr) | 刚好完成圆周运动 |
| 最小角速度 | ω_min = √(g/r) |  |
| 绳张力 | T = m(v²/r - g) | v ≥ √(gr)时成立 |

#### 3.2.2 杆模型（最高点）
| 条件 | 公式 | 说明 |
|------|------|------|
| 最小速度 | v_min = 0 | 杆可提供支持力 |
| 杆的作用力 | N = mg - mv²/r | 可正（支持）可负（拉力） |

#### 3.2.3 轨道模型
| 类型 | 最高点条件 | 说明 |
|------|-----------|------|
| 外轨道 | v ≥ √(gr) | 类似绳模型 |
| 内轨道 | v ≥ 0 | 类似杆模型 |

### 3.3 连接体问题
对于质量分别为 m₁、m₂，半径分别为 r₁、r₂ 的物体：

1. **各自滑动条件**:
   \[
   \omega_1 = \sqrt{\frac{\mu g}{r_1}}, \quad \omega_2 = \sqrt{\frac{\mu g}{r_2}}
   \]

2. **绳出现张力的条件**:
   \[
   \omega_T = \sqrt{\frac{\mu g}{\max(r_1, r_2)}}
   \]

3. **系统滑动条件**（绳连接）:
   \[
   \omega_s = \sqrt{\frac{2\mu g}{r_1 + r_2}}
   \]

## 四、能量关系公式

### 4.1 机械能守恒
对于竖直平面内的圆周运动：
\[
\frac{1}{2}mv_1^2 + mgh_1 = \frac{1}{2}mv_2^2 + mgh_2
\]

### 4.2 特定位置的速度关系
| 关系 | 公式 | 说明 |
|------|------|------|
| 最高点→最低点 | v_low² = v_high² + 4gr | 直径两端 |
| 刚好完成圆周运动 | v_low = √(5gr) | 从最低点到最高点 |

### 4.3 动能与势能转换
\[
\Delta E_k = -\Delta E_p
\]
\[
\frac{1}{2}m(v_2^2 - v_1^2) = mg(h_1 - h_2)
\]

## 五、实际应用公式

### 5.1 汽车转弯
| 情况 | 公式 | 说明 |
|------|------|------|
| 水平路面 | μmg = mv²/R | 靠静摩擦力 |
| 倾斜路面 | mg tanθ = mv²/R | 倾角 θ，无摩擦 |
| 有摩擦倾斜 | 复杂，需解方程 | 摩擦力和支持力合力 |

### 5.2 火车转弯
\[
mg \tan\theta = \frac{mv^2}{R}
\]
\[
v = \sqrt{gR \tan\theta}
\]

### 5.3 飞机俯冲
在最低点：
\[
N - mg = \frac{mv^2}{R}
\]
\[
v = \sqrt{(N/m - g)R}
\]

### 5.4 离心运动
离心加速度：
\[
a = \omega^2 r = \frac{v^2}{r}
\]

## 六、解题常用推导

### 6.1 向心力方程的建立
1. 确定研究对象
2. 分析受力情况
3. 建立坐标系（指向圆心为正）
4. 列方程：ΣF_向 = mv²/r = mω²r

### 6.2 临界条件分析步骤
1. 确定约束类型（绳、杆、轨道等）
2. 找出极限情况（刚好滑动、刚好脱离等）
3. 列出极限方程
4. 解出临界值

### 6.3 连接体问题处理
1. 隔离分析每个物体
2. 列出各自的向心力方程
3. 通过连接条件（绳张力相等）联系方程
4. 联立求解

## 七、重要不等式

### 7.1 速度范围
1. **绳模型最高点**: v ≥ √(gr)
2. **水平转盘**: v ≤ √(μgr) 或 ω ≤ √(μg/r)
3. **倾斜转弯**: v ≤ √(gR(μ+tanθ)/(1-μtanθ))

### 7.2 角速度范围
1. **绳模型**: ω ≥ √(g/r)
2. **转盘单个物体**: ω ≤ √(μg/r)
3. **转盘连接体**: 分段函数，有多个临界值

### 7.3 力的范围
1. **绳张力**: T ≥ 0（绳不能推）
2. **静摩擦力**: 0 ≤ f ≤ μN
3. **轨道压力**: N ≥ 0（轨道不能拉）

## 八、记忆技巧

### 8.1 口诀记忆
- "绳紧杆松": 绳模型要紧（v ≥ √gr），杆模型可松（v ≥ 0）
- "外紧内松": 外轨道要紧，内轨道可松
- "大先滑": 半径大的先滑动（水平转盘）

### 8.2 对比记忆
| 模型 | 最高点最小速度 | 约束特点 |
|------|---------------|----------|
| 绳模型 | √(gr) | 单向约束（只能拉） |
| 杆模型 | 0 | 双向约束（可拉可推） |
| 外轨道 | √(gr) | 单向约束 |
| 内轨道 | 0 | 双向约束 |

## 九、单位制
- 线速度 v: m/s
- 角速度 ω: rad/s
- 半径 r: m
- 质量 m: kg
- 重力加速度 g: 9.8 m/s²（通常取 10 m/s²）
- 向心力 F: N
- 周期 T: s
- 频率 f: Hz

## 十、典型数值
1. 地球表面: g ≈ 9.8 m/s²
2. 常见绳长: L = 0.5-2 m
3. 常见角速度: ω = 1-10 rad/s
4. 摩擦系数: μ = 0.1-0.8
5. 转弯半径: R = 10-100 m（车辆）
