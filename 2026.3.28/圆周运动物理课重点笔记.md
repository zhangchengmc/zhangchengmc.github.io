# 圆周运动物理课重点笔记

## 课程概述
本PPT主要讲解圆周运动的相关物理概念、公式和应用，包含多个典型例题和选择题。

## 一、基本概念

### 1.1 圆周运动的基本物理量
- **角速度 (ω)**: 描述物体绕圆心转动的快慢
- **线速度 (v)**: v = ωr
- **向心加速度 (a)**: a = ω²r = v²/r
- **向心力 (F)**: F = mω²r = mv²/r

### 1.2 临界条件
- **最大静摩擦力**: f_max = μN
- **临界角速度**: 物体开始滑动时的角速度
- **临界线速度**: 物体开始滑动时的线速度

## 二、典型问题类型

### 2.1 水平转盘上的物体
**特点**: 物体靠静摩擦力提供向心力

**关键点**:
1. 静摩擦力随角速度增大而增大
2. 当静摩擦力达到最大值时，物体开始滑动
3. 滑动条件: f_max = mω²r

**例题1** (第10页):
- 两个质量相同的小木块放在转盘上
- 距离转轴距离不同: r_A = l, r_B = 2l
- 最大静摩擦力 f_max = kmg
- 判断哪个先滑动、摩擦力关系等

### 2.2 竖直平面内的圆周运动
**特点**: 重力参与提供向心力

#### 2.2.1 绳模型（无支撑）
**临界条件**: 最高点最小速度 v_min = √(gr)
- 最高点: mg + T = mv²/r
- 最低点: T - mg = mv²/r

**例题2** (第25-26页):
- 小球在轨道最高点刚好不脱离轨道
- "水流星"问题
- 绳的张力分析

#### 2.2.2 杆模型（有支撑）
**特点**: 最高点速度可以为零
- 最高点: 杆可以提供支持力或拉力
- 临界速度: v = 0

**例题3** (第27页):
- 细线与轻杆连接的区别
- 最高点受力分析

### 2.3 连接体问题
**特点**: 多个物体通过绳连接

**例题4** (第19-20页):
- 两个木块用细绳连接
- 绳从无张力到有张力的过程
- 摩擦力变化分析

## 三、重要公式总结

### 3.1 向心力公式
1. F = mω²r
2. F = mv²/r
3. F = 4π²mr/T²

### 3.2 临界速度
1. 水平转盘: ω_critical = √(μg/r)
2. 绳模型最高点: v_min = √(gr)
3. 杆模型最高点: v_min = 0

### 3.3 超重与失重
1. 最低点: 超重，N = mg + mv²/r
2. 最高点: 
   - 绳模型: 可能失重，T = mv²/r - mg
   - 杆模型: 可能超重、失重或正常

## 四、解题技巧

### 4.1 分析步骤
1. 确定研究对象
2. 分析受力情况
3. 建立向心力方程
4. 考虑临界条件
5. 求解并验证

### 4.2 常见错误
1. 混淆绳模型和杆模型的临界条件
2. 忽略静摩擦力的变化
3. 错误判断连接体的运动状态

## 五、典型例题解析

### 5.1 例题01 (第25页)
**题目**: 小球在轨道最高点刚好不脱离轨道
**解析**: 
- 绳模型: 最高点最小速度 v = √(gr)
- 此时轨道压力为零
- 向心力完全由重力提供: mg = mv²/r

### 5.2 例题02 (第26页)
**题目**: "水流星"问题
**解析**:
- 最高点速度 v = √(gr) 时，水不会流出
- 此时绳张力为零，容器底压力为零
- 处于完全失重状态，但仍有重力作用

### 5.3 例题07 (第19-20页)
**题目**: 两个木块用绳连接在转盘上
**解析**:
- 分析绳从无张力到有张力的过程
- 确定各个阶段的角速度范围
- 分析摩擦力的变化规律

## 七、学习建议

1. **理解物理本质**: 不要死记公式，理解向心力的来源
2. **掌握临界分析**: 学会分析各种临界条件
3. **多做典型例题**: 通过练习掌握解题思路
4. **注意区别**: 区分绳模型、杆模型、转盘模型的不同
5. **联系实际**: 将理论应用于实际问题的分析
