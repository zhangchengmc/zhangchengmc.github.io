## 学习路线图

### 第一阶段：基础概念（1-2天）
1. **理解基本物理量**
   - 线速度 v、角速度 ω、周期 T、频率 f
   - 掌握换算关系：v = ωr, ω = 2π/T
   
2. **掌握向心加速度**
   - 公式：a = v²/r = ω²r
   - 方向：始终指向圆心
   - 物理意义：速度方向变化快慢的量度

3. **理解向心力**
   - 不是独立存在的力
   - 是其他力的合力在法向的分量
   - 公式：F = ma = mv²/r = mω²r

### 第二阶段：基本模型（3-4天）
1. **水平转盘模型**
   - 静摩擦力提供向心力
   - 临界条件：f_max = μmg = mω²r
   - 滑动条件：ω > √(μg/r)

2. **绳模型（竖直平面）**
   - 最高点临界速度：v_min = √(gr)
   - 绳只能提供拉力，不能提供推力
   - 能量关系：机械能守恒

3. **杆模型（竖直平面）**
   - 最高点最小速度：v_min = 0
   - 杆可提供拉力或支持力
   - 双向约束特点

### 第三阶段：复杂问题（5-7天）
1. **连接体问题**
   - 水平转盘上的连接体
   - 竖直平面内的连接体
   - 绳张力的阶段性变化

2. **综合应用题**
   - 汽车转弯问题
   - 过山车设计
   - 杂技表演分析

3. **能量综合分析**
   - 机械能守恒与圆周运动结合
   - 摩擦损耗的影响
   - 变半径圆周运动

## 重点难点突破

### 难点1：临界条件分析
**问题表现**: 分不清不同模型的临界条件

**突破方法**:
1. **分类记忆**
   - 绳模型：v_min = √(gr)（最高点）
   - 杆模型：v_min = 0（最高点）
   - 转盘模型：ω_max = √(μg/r)

2. **物理理解**
   - 绳模型：绳只能拉，不能推
   - 杆模型：杆可拉可推
   - 静摩擦力：有最大值限制

3. **图形辅助**
   - 绘制 v-ω 关系图
   - 标注临界点
   - 分析各区域物理意义

### 难点2：连接体问题
**问题表现**: 不会分析多个物体的相互作用

**突破方法**:
1. **隔离分析法**
   - 对每个物体单独分析
   - 列出各自的向心力方程
   - 通过连接条件联系方程

2. **阶段性分析**
   - 确定各个临界角速度
   - 分析每个阶段的特点
   - 注意力的变化规律

3. **数值验证**
   - 代入具体数值计算
   - 检查结果的合理性
   - 理解物理意义

### 难点3：能量与运动结合
**问题表现**: 不会将机械能守恒与圆周运动结合

**突破方法**:
1. **明确适用条件**
   - 只有重力做功时机械能守恒
   - 有摩擦时机械能不守恒

2. **建立方程**
   - 位置1：E₁ = ½mv₁² + mgh₁
   - 位置2：E₂ = ½mv₂² + mgh₂
   - 守恒时：E₁ = E₂

3. **结合向心力方程**
   - 在特定位置列向心力方程
   - 联立能量方程求解

## 典型例题精讲

### 例题类型1：水平转盘单个物体
**题目特征**: 物体放在水平转盘上，靠静摩擦力提供向心力

**解题步骤**:
1. 分析受力：重力、支持力、静摩擦力
2. 向心力方程：f = mω²r
3. 临界条件：f_max = μmg = mω²_max r
4. 求解：ω_max = √(μg/r)

**易错点**: 忘记静摩擦力有最大值

### 例题类型2：绳模型最高点
**题目特征**: 绳系物体在竖直平面做圆周运动

**解题步骤**:
1. 最高点受力：重力、绳张力（向下）
2. 向心力方程：mg + T = mv²/r
3. 临界条件：T = 0 ⇒ v_min = √(gr)
4. 能量关系：从最低点到最高点机械能守恒

**易错点**: 认为绳在最高点可以没有张力

### 例题类型3：连接体问题
**题目特征**: 多个物体通过绳连接在转盘上

**解题步骤**:
1. 隔离分析每个物体
2. 列出各自的向心力方程
3. 通过绳张力联系方程
4. 分析各个临界角速度
5. 分阶段讨论

**易错点**: 搞错哪个物体先达到最大静摩擦

## 常见问题解答

### Q1: 向心力是真实存在的力吗？
**A**: 不是。向心力是效果力，是其他力的合力在法向的分量。它描述的是使物体做圆周运动需要的力。

### Q2: 为什么绳模型最高点要有最小速度？
**A**: 因为绳只能提供拉力，不能提供推力。在最高点，如果速度太小，重力大于所需向心力，物体会掉落，绳无法提供向上的力来阻止。

### Q3: 如何判断水平转盘上哪个物体先滑动？
**A**: 比较它们的 ω_max = √(μg/r)。半径 r 越大的物体，ω_max 越小，所以会先滑动。

### Q4: 连接体问题中，绳张力什么时候出现？
**A**: 当某个物体的静摩擦力达到最大值，但向心力需求继续增加时，不足的部分由绳张力补充。

### Q5: 机械能守恒在什么条件下适用？
**A**: 只有重力（或弹力）做功，其他力不做功或做功代数和为零时，机械能守恒。

## 考试技巧

### 1. 审题技巧
- 圈出关键词：临界、刚好、最小、最大
- 判断模型类型：绳、杆、转盘、轨道
- 注意连接方式：绳连接、直接接触

### 2. 解题技巧
- 先定性分析，再定量计算
- 画受力分析图，标出各力方向
- 分阶段讨论复杂问题
- 检查结果的合理性

### 3. 时间管理
- 基础题：3-5分钟/题
- 中等题：5-8分钟/题  
- 难题：8-12分钟/题
- 留出检查时间

- 数量级是否合理
- 临界条件是否考虑
- 能量是否守恒
