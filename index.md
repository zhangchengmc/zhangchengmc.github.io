# 物理必修二：圆周运动重点笔记
**整理日期：2026年03月21日**

## 一、课程概述
圆周运动是曲线运动的重要形式，在日常生活和工程技术中广泛应用。本课程主要研究匀速圆周运动的规律。

## 二、基本概念

### 1. 圆周运动
- **定义**：质点的运动轨迹是圆的运动
- **特点**：速度方向时刻改变，是变速运动

### 2. 匀速圆周运动
- **定义**：线速度大小不变的圆周运动
- **特点**：
  - 速率不变，速度方向时刻变化
  - 是变速运动，不是匀速运动
  - 加速度方向始终指向圆心

### 3. 描述圆周运动的物理量
| 物理量 | 符号 | 定义 | 单位 | 方向 |
|--------|------|------|------|------|
| 线速度 | v | 质点通过的弧长与所用时间的比值 | m/s | 切线方向 |
| 角速度 | ω | 质点转过的角度与所用时间的比值 | rad/s | 右手螺旋定则 |
| 周期 | T | 运动一周所用的时间 | s | 标量 |
| 频率 | f | 单位时间内完成的圈数 | Hz | 标量 |
| 转速 | n | 单位时间内转过的圈数 | r/s, r/min | 标量 |

### 4. 向心加速度和向心力
- **向心加速度(a)**：描述速度方向变化快慢的物理量
  - 大小：a = v²/r = ω²r = (2π/T)²r
  - 方向：始终指向圆心
- **向心力(F)**：产生向心加速度的力
  - 大小：F = mv²/r = mω²r
  - 方向：始终指向圆心
  - 性质：效果力，不是性质力

## 三、重要公式与关系

### 1. 基本关系式
1. **线速度与角速度**：v = ωr
2. **角速度与周期**：ω = 2π/T
3. **周期与频率**：T = 1/f
4. **线速度与周期**：v = 2πr/T
5. **转速与角速度**：ω = 2πn (n的单位为r/s)

### 2. 向心加速度公式
1. a = v²/r
2. a = ω²r
3. a = (2π/T)²r
4. a = (2πf)²r

### 3. 向心力公式
1. F = mv²/r
2. F = mω²r
3. F = m(2π/T)²r
4. F = m(2πf)²r

## 四、常见题型与解题方法

### 1. 基本计算题
**题型特点**：已知部分物理量，求其他物理量

**解题步骤**：
1. 明确已知量和待求量
2. 选择合适的公式
3. 注意单位统一
4. 代入计算

**例题**：
> 一个质点做匀速圆周运动，半径r=0.5m，周期T=2s，求：
> 1. 线速度v
> 2. 角速度ω
> 3. 向心加速度a

### 2. 比较分析题
**题型特点**：比较两个做圆周运动物体的物理量关系

**解题关键**：
1. 找出相同量（如同轴转动、皮带传动等）
2. 应用比例关系
3. 注意传动装置的特点

**常见传动方式**：
- **同轴转动**：角速度相同，线速度与半径成正比
- **皮带传动**：线速度相同，角速度与半径成反比
- **齿轮传动**：线速度相同，角速度与齿数成反比

### 3. 实际应用题
**常见场景**：
1. 汽车转弯
2. 圆锥摆
3. 火车转弯
4. 卫星运动
5. 离心现象

**解题要点**：
1. 受力分析
2. 建立向心力方程
3. 考虑临界条件
4. 注意实际约束

## 五、易错点与注意事项

### 1. 概念理解
- ❌ 匀速圆周运动是匀速运动
- ✅ 匀速圆周运动是速率不变的变速运动

- ❌ 向心力是一种特殊性质的力
- ✅ 向心力是效果力，可以由重力、弹力、摩擦力等提供

### 2. 公式应用
- 注意公式的适用条件
- 注意单位统一（角度制与弧度制）
- 注意物理量的矢量性

### 3. 实际问题
- 汽车转弯：静摩擦力提供向心力
- 火车转弯：外轨略高于内轨，重力和支持力的合力提供向心力
- 圆锥摆：重力和绳子拉力的合力提供向心力
